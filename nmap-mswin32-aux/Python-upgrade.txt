The Python directory contains Python, GTK+, PyGTK, and Py2exe.

Python
  o http://www.python.org/download
  o Tested: python-2.7.3.msi
  o In the installer, select "Install just for me."
  o Uncheck all the optional features in "Customize Python 2.7.3".
  o The following installations will use the same installation directory
    by reading a registry key.
GTK+, PyGTK, PyGObject, PyCairo
  o http://ftp.gnome.org/pub/gnome/binaries/win32/pygtk
  o Tested: 2.24/pygtk-all-in-one-2.24.2.win32-py2.7.msi
Py2exe
  o http://www.sourceforge.net/projects/py2exe/files/
  o Tested: py2exe-0.6.9.win32-py2.7.exe
